## RUN MIGRATIONS
- `pip install poetry`
- `poetry install`
- `poetry shell`
- `make create_database`

## RUN APPLICATION

- `make run_server`
- Visit `http://localhost:8000/docs#/`


## Run Unit Test Cases
- make test_unit
